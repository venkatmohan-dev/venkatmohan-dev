## 🎯 Featured Projects

### 🏆 Multi-Module Communication System for Autonomous Underwater Vehicle (AUV)
**Final Year Project** | 2025 – 2026 | CSIR-National Institute of Oceanography, Goa

A comprehensive embedded communication system designed for underwater autonomous vehicles with real-world field testing and deployment validation.

**Key Components:**
- **RF Module** – Implemented ZMODEM protocol-based bidirectional RF data transfer using FreeWave modules; achieved reliable point-to-point wireless links for underwater infrastructure deployments
- **WiFi Module** – Configured TCP/IP-based encrypted file transfer over MikroTik WiFi router; enabled seamless real-time data sync from AUV field node to shore Linux server
- **GPS Module** – Parsed NMEA sentences from GPS receiver over UART (C); implemented geo-tagged data logging with UTC timestamps for autonomous field trials
- **Acoustic Telemetry** – Tested and interfaced underwater telemetry systems for communication in subaqueous environments
- **System Automation** – Deployed systemd service units for auto-launch RF listener, GPS logger, and WiFi sync daemon on boot with full automation

**Impact:** Successfully validated communication reliability and data integrity across all three modules under real field conditions with zero manual intervention required.

**Tech Stack:** Embedded Linux, C, UART/SPI/I2C, FreeWave RF, GPS/NMEA, WiFi, Systemd, TCP/IP
